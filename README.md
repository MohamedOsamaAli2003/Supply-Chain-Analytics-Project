- Power BI: Interactive dashboards, KPIs, and DAX measures

- Python (Pandas, NumPy)
- Power BI
- DAX

## Key Insights
- Supplier performance evaluation
- Transportation efficiency analysis
- Product demand and stock optimization
- Profitability and cost analysis
